Escola EAD

Projeto de Plataforma de Ensino Online

Problemas a Serem Resolvidos

  • Professores, escolas e pequenos empreendedores têm dificuldade em criar plataformas acessíveis para ensinar e vender cursos online.
  • Falta de controle de progresso e entrega estruturada de conteúdo.
  • Necessidade de uma solução acessível, organizada e escalável.

Visão geral

O projeto tem como objetivo criar uma plataforma EAD (Ensino a Distância) onde alunos possam se cadastrar, se inscrever em cursos, assistir videoaulas, responder avaliações, gerar certificados e acompanhar seu progresso por trilhas de aprendizagem.

Funcionalidades do MVP/Usuario

  • Cadastro/Login
  • Visualização de cursos disponíveis
  • Inscrição em cursos
  • Emissão de certificados
  • Acompanhamento de progresso

Funcionalidades do MVP/Admin

  • CRUD de cursos
  • CRUD de aulas
  • Upload de videoaulas
  • Emissão de certificados
  • Visualização de relatórios

Stack de Tecnologias

  • Backend: Node.js + Express
  • Frontend (Admin): Admin.js/Prototipação
  • Banco de Dados: MySQL/MariaDB (ORM Sequelize)
  • Testes Automatizados: Jest + Supertest
  • Gestão e Documentação: GitHub Projects + Swagger
