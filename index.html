Escola EAD

Projeto de Plataforma de Ensino Online

Problema a Ser Resolvido

  • Professores, escolas e pequenos empreendedores têm dificuldade em criar plataformas acessíveis para ensinar e vender cursos online.
  • Falta de controle de progresso e entrega estruturada de conteúdo.
  • Necessidade de uma solução acessível, organizada e escalável.

Modelagem do Banco de Dados

  • Banco de dados relacional: MySQL/MariaDB
  • Estrutura com entidades principais:
    • Usuários (alunos e admins)
    • Cursos
    • Módulos e Aulas
    • Inscrições e Progresso
    • Certificados
  • Documentação já concluída (Swagger/Mermaid)

Exemplos do código consumindo o Banco

Stack de Tecnologias

  • Backend: Node.js + Express
  • Frontend (Admin): Admin.js
  • Banco de Dados: MySQL/MariaDB (ORM Sequelize)
  • Testes Automatizados: Jest + Supertest
  • Gestão e Documentação: GitHub Projects + Markdown/Swagger
